# knights-tour
MAT 385 final project

Finds and displays a knight's tour for an m x n board based on warnsdorff's heuristic.

install:<br>
```
git clone https://github.com/htwight/knights-tour
cd knights-tour/src
javac *.java
```
Execute:<br>
```
java KnightsTour
```
